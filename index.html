git tag

给项目打标签, 作用是标记一个点为一个版本号，如v1.0.0,在程序开发到一个阶段后，我们需要打个标签，发布一版本。


$ git tag -a 0.1.0 -m "version 0.1.0"   //打标签
$ git push origin --tags  //提交到服务器上
$ git tag -d 0.1.0     //删除标签
